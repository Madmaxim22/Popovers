# Popovers

![Test and Deploy](https://github.com/Madmaxim22/Popovers/workflows/Test%20and%20Deploy/badge.svg)

Простая реализация popover-элементов на чистом JavaScript без использования внешних библиотек.

## Описание

Данный проект предоставляет легковесную реализацию всплывающих подсказок (popover), которая позволяет отображать дополнительную информацию рядом с элементами управления. Реализация включает поддержку различных позиций (сверху, справа, снизу, слева) и корректно работает с прокруткой страницы.

## Особенности

- Легковесная реализация без зависимостей
- Поддержка различных позиций: top, right, bottom, left
- Автоматическое закрытие при клике вне области или по клавише Escape
- Корректная работа с прокруткой страницы
- Адаптивное позиционирование
- Поддержка заголовка и содержимого popover
- Современная архитектура с использованием модулей ES6

## Использование

### HTML

Для создания popover добавьте к элементу следующие атрибуты:

```html
<button class="popover-trigger" 
        data-toggle="popover" 
        data-title="Заголовок" 
        data-content="Содержимое popover" 
        data-placement="top">
    Триггер
</button>
```

Доступные значения для `data-placement`: `top`, `right`, `bottom`, `left`.

### JavaScript

Инициализация popover:

```javascript
import Popover from './src/js/Popover.js';

const popover = new Popover();
```

## API

### Класс Popover

#### Методы

- `show(trigger)` - показать popover для указанного триггера
- `hide(trigger)` - скрыть popover для указанного триггера
- `hideAll()` - скрыть все открытые popover
- `toggle(trigger)` - переключить состояние popover для указанного триггера
